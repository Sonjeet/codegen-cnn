# Codegen-CNN

C++ framework for composing a CNN via a YAML config file.

# Prerequisites
- Install the dependencies from Brewfile before trying to build project
- launch python venv by `source venv/bin/activate` 

# Useful commands
#### Build the project
```
./build.py
```

#### Run executable
```
./build.py -t
```

#### Run unit tests
```
./build.py -t
```
